# Security Policy

## Reporting Security Issues

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability in any code published here, please report it via:

- Email: ryan.sharpnack.cs@gmail.com
- Include: Detailed description, reproduction steps, potential impact

## Responsible Disclosure

All vulnerability research in this repository follows coordinated disclosure procedures through CISA ICS-CERT.

## Scope

This repository contains:
- Educational security tools
- Proof-of-concept code (sanitized, post-disclosure)
- Research methodologies

All content is intended for authorized security testing only.

## Legal Notice

Unauthorized access to industrial control systems is illegal. All demonstrations are performed in isolated lab environments with explicit authorization.
